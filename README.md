# AI SkillVerse 2.0 🚀

**AI SkillVerse 2.0** is a flagship, full-stack AI learning platform designed to revolutionize how students master programming, data structures, and artificial intelligence. Built with a modern React architecture and powered by state-of-the-art AI models, this platform offers a deeply personalized, interactive learning experience.

![AI SkillVerse Banner](https://via.placeholder.com/1200x400.png?text=AI+SkillVerse+2.0+-+Master+the+Future)

## ✨ Key Features

- 🗺️ **AI Roadmap Generator**: Dynamically generates personalized, step-by-step learning timelines based on your current skill level and target role.
- 🎙️ **AI Voice Mentor (Ram)**: A highly interactive chat mentor powered by OpenRouter API, featuring native browser Speech Synthesis to speak answers out loud.
- 📄 **ATS Resume & Portfolio Builder**: Generate professional, 2-column minimalist resumes with a live preview and direct-to-PDF high-resolution printing.
- 💻 **AI-Powered Code Playground**: Write code in Python, Java, C++, and JavaScript. The AI acts as a strict compiler, evaluating logic and returning simulated outputs with precise error hints.
- 🎯 **Interactive Quizzes**: Test your knowledge across various domains with dynamic scoring and immediate feedback.
- 🏢 **Company-Specific PYQs**: Access a curated database of Previous Year Questions (PYQs) from top tech companies to prep for technical rounds.
- 🏆 **Leaderboard System**: Gamified ranking system to track your progress and compete with peers globally.
- 📰 **Daily AI News & Updates**: Stay ahead of the curve with the latest trends and breakthroughs in generative AI and tech.

## 🛠️ Tech Stack

- **Frontend**: React.js, Tailwind CSS, Framer Motion (for fluid animations)
- **Icons & UI**: Lucide React
- **AI Integration**: OpenRouter API (Access to advanced LLMs)
- **Routing**: React Router DOM
- **Build Tool**: Vite

## 🚀 Getting Started

Follow these steps to run AI SkillVerse 2.0 locally on your machine.

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn
- An [OpenRouter API Key](https://openrouter.ai/)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/hemalatha24088-glitch/AI-SkillVerse-2.0.git
   cd AI-SkillVerse-2.0
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Configure API Keys**
   - Open `src/pages/AIMentor.jsx` and `src/pages/CodePlayground.jsx` (or your `.env` file if configured).
   - Insert your OpenRouter API Key in the designated constant.

4. **Run the Development Server**
   ```bash
   npm run dev
   ```
   Open `http://localhost:5173` in your browser to start learning!
